教学反馈智能分析平台知识库

一、平台概述
1. 本平台用于高校教学反馈的采集、分析和改进建议生成，形成「反馈采集 → 数据分析 → 改进建议」的完整闭环。
2. 教师可查看名下课程的反馈汇总、各维度评分趋势、聚类分析结果和自动生成的改进建议。
3. 学生可提交对课程的匿名反馈，反馈内容对教师端匿名展示（不显示具体学生信息）。
4. 系统的 AI 助手基于通义千问（qwen-max）大模型，具备 Function Calling 能力，可调用工具查询数据库中的反馈数据。

二、反馈维度定义
平台将教学评价分为六个维度，每个维度评分范围为 1-5 分：
1. KNOWLEDGE（知识深度）：教学内容是否深入、概念讲解是否清晰、示例是否有助于理解。
2. PACE（教学节奏）：课程进度是否适中、是否有足够时间消化内容、是否留出提问时间。
3. INTERACTION（交流互动）：课堂互动机会是否充足、是否有提问环节、小组讨论和课堂练习。
4. CLARITY（表达清晰度）：教师表达是否清楚、是否善用类比和实例、课件是否易于理解。
5. ENGAGEMENT（课堂参与）：学生听课投入程度、注意力维持情况、课堂氛围是否积极。
6. MATERIAL（教学材料）：课件质量、预习资料、课后习题是否匹配教学内容、难度梯度是否合理。

三、评分标准
1. 1 分（很差）：该维度存在严重问题，需要重大改进。
2. 2 分（较差）：该维度存在较多不足，改进空间大。
3. 3 分（一般）：该维度基本合格，但有可改进之处。
4. 4 分（良好）：该维度表现较好，只有少数可优化点。
5. 5 分（优秀）：该维度表现出色，可作为教学示范。

四、反馈提交说明
1. 学生登录后可通过聊天界面提交反馈，AI 助手会引导完成反馈流程。
2. 每次反馈需要指定课程、反馈维度和评分（1-5），可附加文字评论。
3. 建议在每次课后及时提交反馈，以帮助教师持续改进教学。
4. 反馈是匿名的：教师端只能看到各维度的评分统计和匿名评论文字，无法追溯到具体学生。
5. 学生可以查看自己提交的历史反馈记录。

五、数据分析说明
1. 聚类分析：系统自动对反馈数据进行聚类，识别学生在各知识维度的掌握程度，找出共性薄弱环节。
2. 趋势分析：按日期范围展示各维度评分的变化趋势，帮助教师了解教学效果的时间变化。
3. 对比分析：可对比教师名下多门课程的反馈数据，找出教学亮点和差距。
4. 异常检测：自动检测评分异常波动（如某维度评分突然大幅下降），及时提醒教师关注。
5. 参与率分析：统计过去 N 周内的反馈参与率变化，评估学生对反馈活动的积极性。
6. 关键词搜索：可按关键词搜索历史反馈的文字内容，快速定位特定问题的反馈。

六、改进建议
1. 系统根据聚类分析结果自动生成针对性的教学改进建议，按 HIGH/MEDIUM/LOW 三个优先级排序。
2. HIGH 优先级：某维度平均分低于 2.5 分，或连续下降，需立即关注。
3. MEDIUM 优先级：某维度平均分在 2.5-3.5 之间，建议持续优化。
4. LOW 优先级：某维度平均分在 3.5 分以上，推荐精益求精。
5. 改进建议的来源可以是 AI 自动生成（generated_by: 'AI'），也可以是教师手动添加（generated_by: 'TEACHER'）。
6. 教师可以在 AI 生成的建议基础上进行修改和补充。

七、数据隐私说明
1. 教师只能查看自己名下课程的反馈数据，无法越权查看其他教师的课程。
2. 学生只能查看自己提交的反馈记录，无法查看其他学生的反馈。
3. 学生端提交反馈时，教师端不会显示具体提交者的信息。
4. 系统记录反馈提交时间，用于趋势分析和参与率统计，但不向教师暴露提交者身份。

八、常见问答
1. 问：如何提交课程反馈？答：登录学生端后，通过聊天界面告诉AI你要提交反馈，AI会引导你完成。
2. 问：反馈是匿名的吗？答：是的，教师端看不到提交者的具体信息，只看到汇总数据和匿名评论。
3. 问：如何查看已选课程？答：学生登录后，AI助手会自动列出您已选的全部课程。
4. 问：忘记密码怎么办？答：请联系院系教务管理员重置密码，系统暂不支持自助重置。
5. 问：反馈数据多久更新一次？答：反馈提交后实时更新，教师端可立即查看分析结果。
6. 问：教师如何查看课程反馈？答：教师登录后，可以通过聊天界面查询名下课程的反馈汇总、趋势、聚类分析等。
7. 问：为什么要提交反馈？答：反馈帮助教师了解教学效果，及时发现和解决问题，持续提升教学质量。

九、系统使用建议中的推荐表达
1. 回答关于反馈提交的问题时，可以表述为：请先确认您要反馈的课程，然后告诉我反馈维度和评分（1-5分），您可以附上您的具体意见。
2. 回答关于教学质量分析的问题时，可以表述为：根据反馈数据，课程在某个维度表现较好（或需要改进），具体数据如下……
3. 回答关于改进建议的问题时，可以表述为：建议针对某薄弱环节采取以下措施……
4. 回答匿名性相关问题时，可以表述为：请放心，您的反馈是匿名的，教师只能看到统计数据，不会看到您的身份信息。
